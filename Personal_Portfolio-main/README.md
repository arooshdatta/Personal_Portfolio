# Personal Portfolio

A modern personal portfolio built with React, Tailwind CSS, and Vite. This project showcases professional experience, projects, skills, achievements, and contact details with a polished interactive UI.

## Features

- Responsive design optimized for desktop and mobile
- Smooth section scrolling and command palette navigation
- Project showcase cards with repository links
- Custom animated UI elements and interactive user experience
- Contact section with email, phone, and social links

## Tech Stack

- React
- Vite
- Tailwind CSS
- Lucide React icons
- ESLint for code quality

## Local Installation

1. Clone the repository:

```bash
git clone https://github.com/arooshdatta/Personal_Portfolio.git
```

2. Install dependencies:

```bash
npm install
```

3. Start the development server:

```bash
npm run dev
```

4. Open the app in your browser at the local URL shown in the terminal.
